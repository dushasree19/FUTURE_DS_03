## Dataset
Public text feedback dataset adapted as college event feedback.

## Approach
- Loaded and explored feedback data
- Cleaned text (lowercasing, removing special characters)
- Applied sentiment analysis using TextBlob
- Visualized sentiment distribution
- Derived actionable insights

## Key Insights
- Majority of feedback is positive, indicating good event reception
- Neutral feedback suggests scope for improvement
- Negative feedback highlights specific concerns to address
- Sentiment analysis enables quick understanding of large feedback data
